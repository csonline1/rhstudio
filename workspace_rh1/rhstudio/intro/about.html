ABOUT rhstudio STUDIO
rhstudio Studio is a development environment used by modelers (developers and business experts) to graphically define the functionality of applications.
It also contains an embedded application server (Tomcat) and an embedded database server(HSQL DB), which allow for instantaneous automatic execution of the modeled applications.

Getting Started with the rhstudio Tutorial
The rhstudio Tutorial provides a step-by-step, hands-on example of the development of a complete sample application. Following the tutorial, it will only take you a few minutes to have a first web application up and running.

The tutorial is available in two forms:
  1. An on-line version provided with rhstudio Studio (Help->Help Contents->rhstudio Documentation->Tutorial).
  2. A PDF version, which you can also print.

You can use the tutorial in two modes:
  1. Hands On: Read the tutorial while following the hands-on instructions on building the sample application. At each step, as you progress with building the application, you can run it from your browser.
  2. Quick: Just read the tutorial without actually building the sample application. Instead, at each step you can import the sample application provided with the tutorial and run it.
The time you will devote to the tutorial depends, of course, on the mode you choose to read it. We estimate that the quick mode requires ½ to 5 hours (depending how far you follow the tutorial), while the hands-on mode may require at least double the time.

For more details on the tutorial, see http://www.rhstudio.com/#Id=121.
