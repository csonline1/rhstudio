开始你的第二个应用程序在两分钟
1) 开始你的第二个应用程序 - 即刻结果
我们现在启动另一个应用程序，它会作为基础的 rhstudio 教程 (看以下的详细):
  • 切换回 rhstudio 工作室。
  • 选择 文件->新->rhstudio Project 打开e New rhstudio Project 向导。
  • 输入您选择的名称 (e.g. Tutorial) 为您的新项目， 选择 Legacy Navigation (2) 模板，并点击 完成 (3).
  • 点击 视图 (1) 模板在显示类别的选项板中，然后点击 Tutorial system 模型在编辑。
  • 当提示你提供一个名称， 名称视图为 Open Requisitions (2).
  • 双击 Open Requisitions 视图模型，放大它。
  • 点击 按钮 模板在显示类别的选项板中，然后点击 Open Requisitions 视图模型。名称它为 New Requisition (3).
  • 双击 New Requisition 按钮模型，放大它。
  • 点击 弹出 (4) 模板在显示类别的选项板中， 然后点击 New Requisition 按钮模型在编辑，来定位 弹出 (5) 在按钮中。
  • 当提示给弹出一个名字 (默认为 弹出)。 名称它为 Enter New Requisition (6).
  • 双击 Enter New Requisition 弹出模型来放大它。
  • 点击 Label (1) 模板在显示类别的选项板中。 放它到 Enter New Requisition 弹出。 名称它为 Description: (2) (包括 ":")
  • 点击 Text Area (3) 模板在显示类别的选项板中。 放它到 Enter New Requisition 弹出。 名称它为 Description (4)
  • 展开 Footer 模型，然后重命名 OK 按钮为 Submit.
2) 查看结果
  • 保存你的更改 (通过 文件->保存 菜单选项，CTRL+S 捷径或 保存 工具栏按钮).
  • 点击 Launch the application 按钮在工作室的主工具栏中，加载您的应用程序在嵌入的 rhstudio 服务器，并打开一个网页浏览器.
  • 请注意视图 (1), 和按钮 (2) 的名称。
  • 点击 New Requisition (2) 按钮来打开 Enter New Requisition (3) 对话。
  • 请注意所有你编辑的元素出现, 包括 Description: (4) label, Description (5) text area, 和 Submit (6) 按钮。
3) 查看项目实例
rhstudio 工作室 是安装了示例项目，提供模型例子的真实世界的应用程序，并能帮助你提高造型技巧。
要查看和定制示例模型:
  • 切换回 Repository Explorer(1) 视图。
  • 双击(2) 任何应用程序来打开在模型编辑器 (3). 
要看到示例行动 - 使用 Launch the application (4) 按钮在工作室的主工具栏中。
4) 下一步?
我们建议你跟随 rhstudio 教程。 详细的教程资料， 在这里。
请注意，如果你跟着上面的步骤，您的模型应该是类似于第二阶段创建的教程。 欢迎你阅读阶段1和第2来获得更详细的资料，或者你可能想直接继续第3阶段。
